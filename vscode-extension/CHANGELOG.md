# Change Log

All notable changes to the "toon-vscode" extension will be documented in this file.

## [1.0.0] - 2025-11-16

### Added
- Initial release
- Syntax highlighting for `.toon` files
- Support for all TOON tokens (OBJ, LIST, STR, INT, FLOAT, BOOL)
- Comment support with `##`
- Auto-closing brackets and quotes
- Smart indentation
